🚀 Passo a passo para criar o site de casamento no Canva

1. Criar o site

  1. Entre no Canva.

  2. Busque por “Site de casamento” na barra de pesquisa.

  3. Escolha um modelo pronto (ou crie do zero).

2. Estrutura básica do site

Inclua seções como:

  • Capa: nome dos noivos + data.

  • Nossa história: um resumo fofo da trajetória do casal.

  • Cerimônia & Festa: local, data e horário.

  • Confirmação de presença (RSVP): pode ser um link para Google Forms.

  • Lista de presentes: parte onde você organiza os links ou Pix.

3. Criando a lista de presentes

Opções:
🔹 Lista tradicional: faça um design bonito com os nomes dos presentes (ex.: “Jogo de panelas”, “Viagem lua de mel”).
🔹 Ao lado, insira um botão ou link com:

  • O link da loja (se quiser que comprem o item específico).

  • Ou um botão escrito “Presentear com Pix”.

4. Adicionando o Pix

Você pode fazer de 2 formas:

✅ Pix Copia e Cola:

  • Escreva “Chave Pix: [sua chave]”.

  • Coloque um botão “Copiar chave Pix” → direcionando para uma página com a chave escrita.

✅ QR Code do Pix:

  • Gere um QR Code no seu banco.

  • Baixe a imagem e insira no site.

  • Coloque um texto do tipo: “Escaneie o QR Code para nos presentear com Pix ❤️”.

5. Publicando o site

  1. Clique em Compartilhar → Site.

  2. Escolha a opção gratuita do Canva.

  3. Ele vai gerar um link (ex.: https://nomedecasamento.canva.site).

  4. Envie esse link para convidados via WhatsApp ou convite impresso.

✨ Extra: Se quiser algo mais profissional (com confirmação automática de presença e lista integrada com pagamentos), pode usar plataformas como iCasei, Casar.com ou Wedy, mas se a ideia é algo simples e personalizado, o Canva já resolve bem.

Boa pergunta 👌 você quer liberdade total, sem pagar planos nem porcentagem para plataformas prontas. Isso é totalmente possível criando um site do zero. Vou te mostrar o que você precisa:

🔧 O que você precisa para criar um site de casamento do zero (sem taxas)

1. Domínio (opcional, mas recomendado)

  • É o endereço do site, tipo: www.casamentojoaomaria.com.br.

  • Custa em média R$ 40 a 60 por ano (registro.br, Hostinger, GoDaddy etc.).

  • Se não quiser gastar nada, pode usar um subdomínio grátis (ex.: GitHub Pages, Netlify, Vercel, Google Sites).

2. Hospedagem (onde o site fica salvo)

  • Pode ser gratuita em serviços como:

    • GitHub Pages (se você souber mexer um pouco com arquivos HTML).

    • Netlify ou Vercel (super fáceis, arrasta e solta).

  • Ou paga, se quiser mais facilidade (mas não é obrigatório).

3. Ferramenta para construir o site

Você tem três opções:

✅ Fácil e grátis (sem código):

  • Google Sites → simples, mas funciona bem para casamento.

  • Canva → já vimos, dá pra fazer bonito, mas limitado no Pix.

✅ Intermediário (visual e sem taxa):

  • WordPress.org (instalado em hospedagem gratuita/ barata).

  • Você baixa um template gratuito de casamento e só personaliza.

✅ Avançado (código, mas total liberdade):

  • Criar em HTML + CSS + JavaScript (eu posso te gerar um modelo pronto e você só hospeda).

  • A vantagem: não tem taxas, nem porcentagem, e você controla tudo.

  • A desvantagem: precisa mexer em arquivos e enviar para o servidor (mas não é tão difícil).

4. Lista de presentes com Pix

Sem depender de plataformas:

  • Crie uma seção “Lista de Presentes”.

  • Coloque QR Codes do Pix (um para cada presente simbólico) ou apenas a chave Pix.

  • Exemplo:

    • “Jantar romântico – R$100” → QR Code do Pix já configurado.

    • “Viagem Lua de Mel – R$500” → outro QR Code.

  • Assim o convidado escolhe e paga direto para você, sem taxas.

(Se quiser, posso te mostrar como gerar QR Codes de Pix personalizados para cada presente 👀).

5. Publicação

  • Se for HTML, basta jogar os arquivos no GitHub Pages, Netlify ou Vercel → site online grátis.

  • Se for Google Sites, Canva ou WordPress gratuito, basta publicar pelo próprio painel.

📌 Resumindo:

  • Mais fácil: Google Sites (0 custo, mas simples).

  • Mais bonito e criativo: Canva (0 custo, mas limitado no Pix).

  • Mais profissional e livre: HTML + hospedagem gratuita (0 custo e sem taxas).

🚀 Passo a passo para hospedar no GitHub Pages

1. Criar conta no GitHub

  1. Acesse github.com

  2. Clique em Sign up e crie sua conta (é grátis).

2. Criar um novo repositório

  1. Depois de logado, clique no botão New repository.

  2. Nomeie como quiser, exemplo: casamento.

  3. Marque a opção Public.

  4. Clique em Create repository.

3. Enviar os arquivos do site

  1. No repositório criado, clique em Add file → Upload files.

  2. Faça o upload do arquivo index.html que você salvou.

  3. Também suba as imagens (os QR Codes e outras imagens personalizadas).

  4. Clique em Commit changes para salvar.

4. Ativar o GitHub Pages

  1. No seu repositório, clique em Settings.

  2. No menu lateral, vá em Pages.

  3. Em Source, escolha a branch main e a pasta /root.

  4. Clique em Save.

  5. Pronto 🎉 Dentro de alguns minutos, seu site estará online!

5. Acessar seu site

  • O endereço vai ser algo assim:

    arduino
    https://seuusuario.github.io/casamento

👉 Dica extra: se você quiser, pode comprar um domínio baratinho (ex.: joaoemaria.com.br) e apontar para o GitHub Pages. Assim o link fica mais elegante para os convidados.
